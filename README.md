# Welcome to your React project

This is a React application built with Vite, TypeScript, and shadcn/ui components.

## Getting Started

### Use this project locally

1. Clone this repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`

Changes made locally will be reflected when you run the project.

## Development

This project uses:
- Vite for fast development and building
- React with TypeScript for type-safe components
- Tailwind CSS for styling
- shadcn/ui for accessible UI components

## Deployment

To deploy your project:

1. Build the project with `npm run build`
2. Deploy the contents of the `dist` folder to your hosting provider

## Can I connect a custom domain to my project?

Yes, most hosting providers allow you to connect a custom domain to your project. Check your hosting provider's documentation for specific instructions.
